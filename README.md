Insurance Analytics Dashboard | Power BI

Tools: Power BI, DAX, Power Query, Excel

Designed and developed an interactive Insurance Performance Dashboard for PRSIM Insurance Pvt. Ltd. to analyze premium, claims, and policy performance.

Built dynamic KPIs including Total Premium (5.98M), Coverage Amount (600.55M), and Claim Amount (16.91M) for executive-level insights.

Created visual reports to track:

Premium distribution by policy type (Travel, Health, Auto, Life, Home)

Claim amount by age group

Active vs Inactive policy ratio

Claims status breakdown (Pending, Rejected, Settled)

Implemented slicers for Claim Number, Policy Number, and Customer ID to enable drill-down and real-time filtering.

Used DAX measures for calculated fields, aggregations, and percentage analysis.

Performed data transformation and cleaning using Power Query.

Delivered actionable insights to improve claim settlement tracking and policy performance monitoring.
